Algorithm:
1. Function signature:
   - Function name: check_string
   - Parameters: S (string)
   - Return type: string

2. Variable declarations:
   - c: Counter (from collections module)
   - Sa: set of characters (set of strings)

3. Required libraries:
   - collections.Counter

4. Execution logic:
   a. Initialize c as a Counter object of S.
   b. Convert the list of characters in S to a set and assign it to Sa.
   c. Check if the length of Sa is not equal to 2.
      - If true, return "No".
   d. Iterate over each character i in Sa.
      i. Check if the count of i in c is not equal to 2.
         - If true, return "No".
   e. If all checks are passed, return "Yes".
