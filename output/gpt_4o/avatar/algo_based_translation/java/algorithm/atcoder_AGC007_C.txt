```
Function: main()
    Dependencies: java.math.BigDecimal, java.text.DecimalFormat, java.util.Scanner
    Variables:
        scan: Scanner
        N: double
        d: double
        x: double
        ans: BigDecimal
        adnum: BigDecimal
        format: DecimalFormat
        
    Begin
        Initialize scan with new Scanner(System.in)
        Read N from input using scan.nextDouble()
        Read d from input using scan.nextDouble()
        Read x from input using scan.nextDouble()
        Initialize ans with new BigDecimal("0.0")
        
        While N > 0.5
            Calculate adnum as BigDecimal.valueOf(d + x * (N - 0.5))
            Add adnum to ans
            Update d with (N + 1.0) * d / N + (5.0 * x) / (2.0 * N)
            Update x with (1.0 + (2.0 / N)) * x
            Decrement N by 1
            
        Initialize format with new DecimalFormat("#.#")
        Set minimum fraction digits of format to 20
        Print formatted ans using format.format(ans)
    End
```
