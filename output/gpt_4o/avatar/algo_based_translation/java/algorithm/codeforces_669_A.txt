   - Parameters: `String[] subhani` (array of strings)
   - Return type: `void`
   - Variables:
     - `BufferedReader br` of type `BufferedReader`

     1. Create a `BufferedReader` object `br` to read input from the standard input stream.

     3. Call the `solve` function with `n` as the argument.

   - Parameters: `int n`
   - Return type: `void`
   - Logic:
     1. Calculate the result using the formula: `(n / 3) * 2 + (n % 3 == 0 ? 0 : 1)`.
     2. Print the result to the standard output.
