1. Import the necessary libraries: `java.lang.Math` and `java.util.Scanner`.

2. Define the main function:
   - Signature: `public static void main(String[] args)`

3. Declare and initialize a `Scanner` object to read input: `Scanner sc = new Scanner(System.in);`

4. Read two integers, `N` and `D`, from input:
   - `int N = sc.nextInt();`
   - `int D = sc.nextInt();`

5. Declare a 2D integer array `vectors` with dimensions `[N][D]`.

6. Populate the `vectors` array with input values:
   - Loop over `i` from 0 to `N-1`:
     - Loop over `j` from 0 to `D-1`:
       - Assign `vectors[i][j] = sc.nextInt();`

7. Declare an integer `answer` and initialize it to 0.

8. Calculate the number of pairs with an integer distance:
   - Loop over `i` from 0 to `N-2`:
     - Loop over `j` from `i+1` to `N-1`:
       - Declare an integer `dist` and initialize it to 0.
       - Loop over `d` from 0 to `D-1`:
         - Compute the difference `x = vectors[i][d] - vectors[j][d]`.
         - Add the square of the difference to `dist`: `dist += (x * x);`
       - Calculate the square root of `dist` and store it in a double `sq`: `double sq = Math.sqrt(dist);`
       - Increment `answer` if `sq` is an integer:
         - Check if `Math.abs(sq - Math.floor(sq)) < 0.001`.
         - If true, increment `answer`: `answer += 1;`

9. Output the result:
   - Print `answer` to the console: `System.out.println(answer);`
