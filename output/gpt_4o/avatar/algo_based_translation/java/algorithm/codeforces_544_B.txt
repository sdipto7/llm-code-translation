   - Declare `int b` and calculate as `n / 2`.
   - Declare `int max` and calculate as `a * a + b * b`.
   - If `count > max`, print "NO" and return.
   - Print "YES".
   - For loop iterating `int i` from `0` to `n-1`:
     - Instantiate `StringBuilder sb`.
     - For loop iterating `int j` from `0` to `n-1`:
