1. Import the following libraries:
   - `java.io.PrintWriter`
   - `java.util.Scanner`

2. Define the main function `public static void main(String[] args)` with no return type.

3. Initialize a `Scanner` object `sc` to read input from the standard input stream.

4. Read the first input value as a `String` using `sc.next()` and parse it to an integer `A` using `Integer.parseInt()`.

5. Read the second input value as a `String` using `sc.next()` and parse it to an integer `B` using `Integer.parseInt()`.

6. Declare an integer variable `amari` and initialize it to 0.

7. Declare an integer variable `result` and initialize it to 0.

8. Calculate the value of `result` using the formula `(B - 1) / (A - 1)` and assign it to `result`.

9. Calculate the remainder `amari` using the formula `(B - 1) % (A - 1)`.

10. Use an `if` statement to check if `amari` is not equal to 0:
    - If true, increment `result` by 1.

11. Initialize a `PrintWriter` object `out` to write output to the standard output stream.

12. Print the value of `result` using `out.println(result)`.

13. Close the `Scanner` object `sc` using `sc.close()`.

14. Flush the `PrintWriter` object `out` using `out.flush()`.
