- Import required module: `java.util.Scanner`
- Define the main function `main` with parameters `args` (array of Strings), return type `void`
- Declare and initialize `Scanner` object `sc` to read input from standard input
- Start an infinite loop:
  - Declare and initialize integer variable `sum` to `0`
  - Read a string `s` using `sc.next()`
  - If `s` is equal to `"0"`, break the loop
  - Iterate over each character in the string `s` with an index `i` ranging from `0` to `s.length() - 1`:
    - Get the character `a` at index `i` using `s.charAt(i)`
    - Add the integer value of character `a` to `sum` by calculating `a - '0'`
  - Print the value of `sum` using `System.out.printf` with format `"%d\n"`
- Loop continues until the condition to break is met
