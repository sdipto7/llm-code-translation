- Import required library: `java.util.Scanner`
- Define function `main` with parameter `args` of type `String[]` and no return type
- Declare variable `sc` of type `Scanner` and initialize it with `new Scanner(System.in)`
- Declare variable `X` of type `int` and assign it the value obtained from `sc.nextInt()`
- Declare variable `t` of type `int` and assign it the value obtained from `sc.nextInt()`
- If condition `(X - t) >= 0` is true:
  - Output the result of `X - t` using `System.out.println`
- Else:
  - Output `0` using `System.out.println`
