Algorithm Main

   a. Declare integer variables:
      - int ans: Initialize to 0, it will store the accumulated result.
   b. For loop with integer i starting from 1; condition (i * d < 600); increment i:
      - int x: Calculate i * d.
      - int y: Calculate x * x.
      - Add y * d to ans.
   c. Return ans.

5. Function main():
   a. While the function read() returns true:
      - Print the result of calling solve().

End Algorithm
