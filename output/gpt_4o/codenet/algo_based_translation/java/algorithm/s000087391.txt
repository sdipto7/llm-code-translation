```
1. Import necessary libraries:
   - Import `java.util.ArrayList`
   - Import `java.util.Scanner`

2. Define the main function `main()` with no parameters and no return type:
   - Declare `ArrayList<Integer> result` to store the output numbers.
   - Initialize `Scanner insert` to read input from the console.
   - Declare integer `n` and set it to -1.

3. Start an infinite loop:
   - Read a line of input using `insert.nextLine()` and store it in `line`.
   - Parse `line` to an integer and assign to `n`.
   - If `n` equals 0, break the loop.

4. Inside the loop:
   - Initialize an integer `sum` to 0.
   - Declare integer array `input` with size `n`.
   - Read another line of input using `insert.nextLine()` and split it by spaces into a string array `array2`.

5. Iterate over the range from 0 to `n-1`:
   - Parse each element of `array2` to an integer and store in `input[i]`.
   - Add `input[i]` to `sum`.

6. Calculate the average:
   - Compute `avg` as `sum` divided by `n` and cast to `double`.

7. Initialize an integer `num` to 0.

8. Iterate over the range from 0 to `n-1`:
   - If `input[i]` is less than or equal to `avg`, increment `num` by 1.

9. Add `num` to the `result` list.

10. After the loop, iterate over each element in `result`:
    - If the element is equal to -1, print "NONE".
    - Otherwise, print the element.

11. End of the main function.
```
