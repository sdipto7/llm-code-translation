- Define a class `Main`.

- Declare a `Scanner` object `scanner` and initialize it with `new Scanner(System.in)`.
- Declare an integer variable `h` and assign it the next integer input from `scanner`.
- Declare an integer variable `a` and assign it the next integer input from `scanner`.
- Check if the remainder of `h` divided by `a` is equal to 0:

  - If false, print the result of integer division `h / a` plus 1.
