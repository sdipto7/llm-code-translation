Algorithm:

1. Import the Scanner class from the java.util package.

2. Define the main function with the signature:
   - `public static void main(String[] args)`

3. Within the main function, execute the following steps:

   a. Declare a Scanner object for input operations:
      - `Scanner scan`

   b. Read an integer input from the user:
      - `int n = scan.nextInt()`

   c. Initialize an integer variable `count` to 0:
      - `int count = 0`

   d. Execute a for loop with the following structure:
      - `for (int i = 1; i <= n; i++)`
        i. Increment `count` by the current value of `i`:
           - `count += i`

4. Print the value of `count` to the standard output:
   - `System.out.println(count)`
