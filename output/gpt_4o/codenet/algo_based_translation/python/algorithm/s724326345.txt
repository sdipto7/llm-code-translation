```
Algorithm main():
    Input: Read a line of input consisting of five integers A, B, C, X, Y.
    Variables:
        A, B, C, X, Y: Integers
        ans: Integer, initially set to 10^10

    Loop k from 0 to max(X, Y) inclusive:
        total = 2 * C * k + A * max(0, X - k) + B * max(0, Y - k)
        If total < ans:
            ans = total

    Output: Print the value of ans

Dependencies: None
```
