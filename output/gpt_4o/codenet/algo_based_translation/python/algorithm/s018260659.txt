1. Function Signature:
   - `def check_product(N: int) -> str`

2. Variable Declarations:
   - `cnt: int = 1`

3. Logic Flow:
   1. Initialize `cnt` to 1.
   2. Begin a while loop with the condition `cnt <= 9`.
      - Check if `N % cnt == 0` and `1 <= N / cnt <= 9`.
        - If both conditions are true, return 'Yes' and exit the loop.
      - If the conditions are not satisfied, increment `cnt` by 1.
   3. If the loop completes without returning, return 'No'.

4. Control Structures:
   - While loop with condition `cnt <= 9`.
   - If condition `N % cnt == 0 and 1 <= N / cnt <= 9`.
   - Else condition to increment `cnt`.

5. Input/Output Operations:
   - Input: `N` is an integer parameter to the function.
   - Output: Returns 'Yes' if a valid pair is found, otherwise 'No'.

6. Libraries, Modules, Dependencies:
   - No external libraries, modules, or dependencies are required.
