Algorithm:

1. Declare a function `calculate_difference(n: int) -> int`.

2. Inside the function, perform the following steps:

   a. Check if `n % 1000 == 0`:
      - If true, return `0`.

   b. If `n % 1000 != 0`:
      - Calculate the difference as `1000 - (n % 1000)`.
      - Return the calculated difference.

3. Declare a main function `main()` with no parameters.

4. Inside the `main()` function, perform the following steps:

   a. Read an integer input from the user and store it in a variable `n` of type `int`.

   b. Call the function `calculate_difference(n)` and store the result in a variable `result` of type `int`.

   c. Print the value of `result`.

5. Execute the `main()` function to start the program.
