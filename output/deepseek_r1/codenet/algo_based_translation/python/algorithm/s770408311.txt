  - Name: `main`

  - `n`: integer (initialized via input)

  - `a`: string (temporary per iteration)  
  - Loop counter `i`: integer (range 0 to n-1)  

- Logic flow:  
  1. Read integer `n` from standard input.  
  2. Initialize empty list `res`.  
  3. Loop `n` times:  
      a. Read string `a` from input.  
      b. Append `a` to `res`.  
  4. Convert `res` to a set to remove duplicates.  
  5. Compute length of the set.  
  6. Print the length.

  - For loop: `for i in range(n)`

  - Read `n` as integer from stdin.
