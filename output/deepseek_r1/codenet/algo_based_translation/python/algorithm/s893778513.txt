  - s: int (sum of all elements in l)
- Logic Flow:
  1. Read n via input()
  2. Read space-separated integers via input() to create list l
  3. Compute b = max(l)
  4. Compute s = sum(l)
  5. Check if b < (s - b)
     - True: print("Yes")
     - False: print("No")

  - Single if-else block with condition (b < (s - b))
