**Algorithm:**

1. **Function Signature:**
   - Function Name: `check_possible`
   - Parameters: `a: int`, `b: int`
   - Return Type: `str`

2. **Variable Declarations:**
   - `a: int` (user input)
   - `b: int` (user input)

3. **Logic Flow:**
   - Read one line of input from standard input.
   - Split the input into two values and convert them to integers `a` and `b`.
   - Check the following conditions in order:
     1. `a % 3 == 0`
     2. `b % 3 == 0`
     3. `(a + b) % 3 == 0`
   - If any condition evaluates to `True`, return `'Possible'`.
   - If all conditions evaluate to `False`, return `'Impossible'`.

4. **Control Structures:**
   - Single conditional block with logical OR operations between three sub-conditions.
   - If-Else structure to determine the output.

5. **Input/Output Operations:**
   - Input: Read via `input().split()` and convert to integers.
   - Output: Print result (`'Possible'` or `'Impossible'`) based on conditions.

6. **Dependencies:**
   - No external libraries/modules required.
   - Uses standard input/output functions.
