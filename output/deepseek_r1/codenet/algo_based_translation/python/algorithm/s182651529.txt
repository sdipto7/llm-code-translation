      - Compare `score[mae]` and `score[usiro]`:
        - If `score[mae] < score[usiro]`: Print "Yes".
        - Else: Print "No".
      - Increment `mae` by 1.
      - Increment `usiro` by 1.

  - `for` loop over `range(N - K)` (Python syntax).
  - `if-else` conditional with exact condition `score[mae] < score[usiro]`.
