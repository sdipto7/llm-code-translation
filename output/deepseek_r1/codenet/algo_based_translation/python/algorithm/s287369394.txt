- **Input Operations**:
  1. Read integer `N` from input.
  2. Read list `A` of `N + 1` integers from input.
  3. Read list `B` of `N` integers from input.

- **Variable Declarations**:
  - `N: int`
  - `A: List[int]` (length `N + 1`)
  - `B: List[int]` (length `N`)
  - `ans: int` (initialized to `0`)
  - `d: int` (temporary variable in loop)

- **Logic Flow**:
  1. Initialize `ans = 0`.
  2. Iterate `i` from `0` to `N - 1` (inclusive):
     1. Compute `d = min(A[i], B[i])`.
     2. Add `d` to `ans`.
     3. Subtract `d` from `A[i]` and `B[i]`.
     4. Compute `d = min(A[i + 1], B[i])`.
     5. Add `d` to `ans`.
     6. Subtract `d` from `A[i + 1]` and `B[i]`.
  3. Print `ans`.

- **Control Structures**:
  - A single `for` loop over `range(N)`.
  - Two sequential `min` operations per iteration.
  - Arithmetic updates to `A`, `B`, and `ans`.

- **Output Operation**:
  - Print the final value of `ans`.
