**Function Signature:**
- `main() -> void`

**Variables:**
- `A: int`
- `B: int`
- `C: int`
- `X: int`
- `Y: int`
- `ans: int` (initialized to `10**10`)
- `k: int` (loop variable)
- `total: int`

**Input/Output:**
- Read one line of input, split into five integers `A`, `B`, `C`, `X`, `Y`.
- Print integer `ans` as output.

**Logic Flow:**
1. Initialize `ans` to a large value (`10**10`).
2. Iterate `k` from `0` to `max(X, Y)` (inclusive).
3. For each `k`:
   - Calculate `total = 2 * C * k + A * max(0, X - k) + B * max(0, Y - k)`.
   - If `total < ans`, set `ans = total`.
4. After loop completes, print `ans`.

**Control Structures:**
- For loop: `for k in range(0, max(X, Y) + 1):`
- Conditional: `if total < ans: ans = total`

**Dependencies:**
- None (uses standard input/output operations).
