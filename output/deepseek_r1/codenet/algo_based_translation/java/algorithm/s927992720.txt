  int b (second operand)

- Logic flow:  
  1. Initialize Scanner with System.in  
  2. Start infinite loop:  
      a. Read integer a from input  
      b. Read string op from input  
      c. Read integer b from input  
      d. If op equals "?", break loop  
      e. If op equals "+", print (a + b)  
      f. If op equals "-", print (a - b)  
      g. If op equals "*", print (a * b)  
      h. If op equals "/", print (a / b)  
  3. Loop resumes until broken by "?" operator  
  4. Program terminates after loop exit
