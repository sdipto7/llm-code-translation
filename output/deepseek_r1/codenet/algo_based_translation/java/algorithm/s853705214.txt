**Function Signature:**
- `public static void main(String[] args)`

**Input/Output Operations:**
- Import `java.util.Scanner`
- Read integer `n` from standard input
- Read `n` integers into array `nums`
- Print result to standard output

**Variables:**
- `Scanner sc = new Scanner(System.in)`
- `int n`
- `int[] nums`
- `int min = Integer.MAX_VALUE`
- `int countNegatives = 0`
- `long sum = 0`

**Logic Flow:**
1. Initialize input scanner
2. Read integer `n`
3. Declare array `nums` of size `n`
4. Populate `nums` with `n` integers via loop:
   - `for (int i = 0; i < n; i++)`
5. Process each number in `nums`:
   - For each `num` in `nums`:
     - Calculate absolute value `abs = Math.abs(num)`
     - Add `abs` to `sum`
     - Update `min` to smaller of `min` and `abs`
     - If `num < 0`, increment `countNegatives`
6. Determine final result:
   - If `countNegatives % 2 == 0`: output `sum`
   - Else: output `sum - (min * 2)`

**Control Structures:**
- `for` loop with index (array population)
- Enhanced `for` loop (array processing)
- `if` condition (`num < 0` check)
- Ternary operator for final output decision

**Dependencies:**
- Requires `java.util.Scanner` for input handling
