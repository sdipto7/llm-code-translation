- **Function Signature**:  
  `public static void main(String[] args)`  

- **Variables**:  
  - `Scanner sc = new Scanner(System.in)`  
  - `int n`  
  - `String[] array`  
  - `int cnt = 0`  
  - `int cnt2 = 0`  
  - `String a`  

- **Input Operations**:  
  1. Read integer `n` with `sc.nextInt()`  
  2. Initialize `array` with size `n`  
  3. Loop `n` times to read strings into `array`  

- **Data Processing**:  
  1. Sort `array` lexicographically with `Arrays.sort(array)`  
  2. Set `a = array[0]` and `cnt = 1`  
  3. Iterate over each element in `array`:  
     - If `a` != current element (`array[i]`), increment `cnt`  
     - Update `a` to current element (`array[i]`)  

- **Control Structures**:  
  - `for (int i = 0; i < n; i++)` (input reading loop)  
  - `for (int i = 0; i < n; i++)` (processing loop)  
  - `if (a.equals(array[i]))` (equality check with else branch for incrementing `cnt`)  

- **Output Operation**:  
  Print `cnt` with `System.out.println(cnt)`  

- **Dependencies**:  
  `import java.util.*` (for `Scanner` and `Arrays`)
