**Algorithm:**

1. **Input Reading:**
   - Read `n: long`, `a: long`, `b: long` from standard input.

2. **Variable Declarations:**
   - Compute `sa: long = absolute difference between a and b`.
   - Initialize `ans: long = 0`.

3. **Conditional Logic:**
   - **If** `sa % 2 == 0` (even difference):
     - Set `ans = sa / 2`.
   - **Else** (odd difference):
     - Compute:
       - `AAA: long = minimum of (n - a) and (a - 1)`
       - `BBB: long = minimum of (n - b) and (b - 1)`
     - Set `ans = minimum of AAA and BBB`.
     - Decrement `sa: sa = sa - 1`.
     - Increment `ans: ans = ans + 1`.
     - Add `sa / 2` to `ans`.

4. **Output:**
   - Print `ans`.
