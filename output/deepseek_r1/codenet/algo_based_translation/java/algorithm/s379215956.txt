**Algorithm Extraction:**

- **Function Signature:**
  - `public static void main(String[] args)`

- **Variables:**
  - `Scanner sc` (Scanner object)
  - `String s` (input string)
  - `String s1, s2, s3, s4` (substrings of `s`)
  - `int a = 0` (flag variable)

- **Input Operations:**
  - Read input string via `Scanner`: `s = sc.next()`
  - Split `s` into first four characters: `s1 = s.substring(0,1)`, `s2 = s.substring(1,2)`, `s3 = s.substring(2,3)`, `s4 = s.substring(3,4)`

- **Logic Flow:**
  1. Check if `s1` equals `s2` → set `a = 1` if true
  2. Check if `s2` equals `s3` → set `a = 1` if true
  3. Check if `s3` equals `s4` → set `a = 1` if true
  4. If `a == 1` → print "Bad"
  5. Else → print "Good"

- **Control Structures:**
  - Three independent `if` statements for equality checks
  - Final `if-else` to determine output

- **Dependencies:**
  - `java.util.Scanner`

**Python Code:**

```python
s = input().strip()
a = 0

if len(s) >= 1 and s[0] == s[1]:
    a = 1
if len(s) >= 2 and s[1] == s[2]:
    a = 1
if len(s) >= 3 and s[2] == s[3]:
    a = 1

print("Bad" if a == 1 else "Good")
```
