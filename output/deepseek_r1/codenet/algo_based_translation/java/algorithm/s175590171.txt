- Function Signatures:
  - `main(String[] args)`
  - Custom Input Scanner Class (`FastScanner`) with methods: `hasNextByte`, `readByte`, `isPrintableChar`, `hasNext`, `next`, `nextLong`, `nextInt`, `nextDouble`

- Variables:
  - `n: int` (input size)
  - `a: int[]` (input array)
  - `sum: long` (cumulative sum)
  - `count: int` (track valid consecutive elements)

- Input Operations:
  1. Read integer `n` from standard input
  2. Read `n` integers into array `a`

- Processing Logic:
  1. Sort array `a` in non-decreasing order
  2. Iterate through elements 0 to `n-2` (inclusive):
     a. Add `a[i]` to `sum`
     b. Check if `a[i+1] <= 2 * sum`
        - TRUE: increment `count` by 1
        - FALSE: reset `count` to 0
  3. Compute final result as `count + 1`

- Control Structures:
  - `for` loop over `i` from 0 to `n-2`
  - `if-else` conditional during iteration to update `count`

- Output:
  - Print `count + 1` after processing all elements

- Dependencies:
  - Java `Arrays.sort()` for array sorting
  - Custom `FastScanner` for optimized input parsing
  - Java I/O classes (`InputStream`, `IOException`)
