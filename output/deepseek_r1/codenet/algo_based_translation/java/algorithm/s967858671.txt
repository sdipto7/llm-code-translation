**Algorithm Extraction:**

**Functions:**
- `public static void main(String[] args)`  
  (No parameters, returns void)

**Variables:**
- `int[] Num` (size 11)
- `Scanner n` (initialized with `System.in`)
- Loop indices: `int i`, `int a`, `int b`

**Libraries/Dependencies:**
- `java.util.Scanner`

**Input:**
- Read 10 integers via `n.nextInt()` stored in `Num[0..9]`

**Logic Flow:**
1. Initialize `Num` array (11 elements)
2. Input loop (10 iterations):
   - `for (i = 0; i < 10; i++)` → `Num[i] = n.nextInt()`
3. Sorting (Descending order via Bubble Sort):
   - Outer loop: `a` from 9 down to 0
     - Inner loop: `b` from 0 to `a-1`
       - If `Num[b] < Num[b+1]`:
         - Swap using `Num[10]` as temp:
           1. `Num[10] = Num[b]`
           2. `Num[b] = Num[b+1]`
           3. `Num[b+1] = Num[10]`
4. Output:
   - Print `Num[0]`
   - Print `Num[1]`
   - Print `Num[2]`

**Control Structures:**
- Sequential execution in `main`
- Input loop (`for` with fixed 10 iterations)
- Nested sorting loops (outer `a`-loop: 9→0; inner `b`-loop: 0→a-1)
- Conditional swap (`if (Num[b] < Num[b+1])`)
- Three sequential print statements
