- Function signature:
  - `public static void main(String[] args) throws IOException`
  
- Variables:
  - `InputStreamReader reader` initialized with `System.in` and `StandardCharsets.UTF_8`
  - `BufferedReader in` initialized with `reader`
  - `String[] str` initialized via `in.readLine().split(" ")`
  - `int n` initialized from `str[0]`
  - `double d` initialized from `str[1]`
  - `int cnt` initialized to 0
  - Loop variables: `int i` (loop counter), `double x`, `double y` per iteration
  
- Logic flow:
  1. Create input stream reader and buffered reader
  2. Read first input line and split into string array
  3. Parse first array element as integer `n`
  4. Parse second array element as integer, convert to double `d`
  5. Initialize counter `cnt` to 0
  6. Loop: `for i from 0 to n-1 inclusive`
     a. Read subsequent input line and split into string array
     b. Parse first element as `x` (double)
     c. Parse second element as `y` (double)
     d. Calculate `sqrt(x^2 + y^2)`
     e. If result ≤ `d`: increment `cnt`
  7. Print `cnt` after loop completion
  
- Control structures:
  - Single `for` loop with range `0 ≤ i < n`
  - `if` condition: `Math.sqrt(x*x + y*y) <= d`
  
- Input operations:
  - Read from standard input via `BufferedReader`
  - Split input lines by space delimiter
  - Multiple integer parsing operations via `Integer.parseInt()`
  
- Output operations:
  - Single `System.out.println(cnt)` after processing
  
- Dependencies:
  - `java.io.BufferedReader`
  - `java.io.IOException`
  - `java.io.InputStreamReader`
  - `java.nio.charset.StandardCharsets`
