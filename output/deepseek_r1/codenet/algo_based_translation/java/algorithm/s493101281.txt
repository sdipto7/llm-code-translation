- Functions:
  - `public static void main(String[] args)`: Entry point
    Parameters: String array args
    Return: void
  - `private static void print(Object o)`: Output helper
    Parameters: Object o
    Return: void
  - `FastScanner` class methods: nextInt(), nextLong(), nextDouble(), next()
    - Handles input parsing
- Variables (main):
  - `n: Integer` - Input list size
  - `l: List<Integer>` - Stores input numbers
  - Loop counter `i: Integer`
- Logic Flow:
  1. Initialize FastScanner and PrintWriter
  2. Read integer n from input
  3. Read n integers into list l
  4. Sort list l in ascending order
  5. Compute difference between elements at indices (n//2) and (n//2 -1) in sorted list
  6. Print computed difference
- Control Structures:
  - `for (i=0; i<n; i++)` - Input reading loop
  - Implicit array index calculation via integer division
- Data Structures:
  - ArrayList for number storage
- I/O Operations:
  - Input: Uses FastScanner to read integers
  - Output: Uses System.out.println via print() helper
- Dependencies:
  - Requires java.util.* (Collections, List, ArrayList)
  - Requires java.io.* (InputStream, PrintWriter)
  - Internal FastScanner class for optimized input handling
- Critical Path:
  Input → Sort → Calculate median difference → Output
