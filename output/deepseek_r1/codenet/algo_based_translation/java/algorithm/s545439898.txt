# Algorithm Extraction

**Function Signatures:**
- `public static void main(String[] args)`

**Variables:**
- `Scanner sc` (type: `Scanner`)
- `int count` (initialized via `sc.next()`)
- `int[] givingNumbers` (size: `count`)

**Libraries/Dependencies:**
- `java.util.Scanner`

**Logic Flow:**
1. Initialize `sc` with `System.in`
2. Read `count` as integer from input
3. Declare `givingNumbers` array with size `count`
4. For loop: `i` from `0` to `count-1` (inclusive):
   - Read next input token, parse to integer, store in `givingNumbers[i]`
5. For loop: `i` from `count-1` down to `0` (inclusive):
   - Print `givingNumbers[i]`
   - If `i > 0`:
     - Print space character
6. Print newline after reversing loop completes
7. Close `sc`

**Control Structures:**
- Loop: `for (int i = 0; i < count; i++)` for input reading
- Loop: `for (int i = count-1; i >= 0; i--)` for reversal/output
- Conditional: `if (i > 0)` controlling space printing

**Input/Output Operations:**
- Input: Sequential integer parsing via `sc.next()`
- Output: Space-separated reversed array elements followed by newline
