```python
Function: main()
Parameters: None (command-line arguments via system)
Return Type: void

Variables:
- x: int (initialized to 0)
- y: int (initialized to 0)
- t: int (initialized to 0)
- n: int (input from stdin)
- str: String (input from stdin)
- l: char[] (converted from str)

Algorithm Steps:
1. Read n from stdin
2. Read str from stdin
3. Convert str into a char array l
4. Loop i from 0 to n-1:
   a. If l[i] is not '4' and not '7':
      i. Set t = 1
5. If t == 1:
   a. Print "NO"
6. Else:
   a. Loop i from 0 to (n//2) - 1:
      i. Add numeric value of l[i] to x
   b. Loop i from n-1 downto (n//2):
      i. Add numeric value of l[i] to y
   c. If x == y:
      i. Print "YES"
   d. Else:
      i. Print "NO"

Control Structures:
- For loop with explicit index ranges
- Conditional checks for character validity
- Two separate sum accumulations for first and second halves
- Final comparison of sums for result

Input/Output:
- Reads n and str via Scanner
- Prints "NO" if invalid characters or sum mismatch
- Prints "YES" if valid characters and equal sums

Dependencies:
- Requires java.util.Scanner
- Uses Character.getNumericValue() for conversion
```
