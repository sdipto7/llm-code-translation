**Algorithm Extraction:**

**Functions:**
- `public static void main(String[] args)`
- `static void swap(int a, int b)`
- `static void debug(Object... obj)`

**Variables:**
- `int a`, `int b` (input values)
- `int c = 0` (counter)

**Input/Output:**
- Read `a` and `b` via `Scanner`.
- Output via `PrintWriter`.

**Logic Flow:**
1. If `a == 1 && b == 1`:  
   - Print `0` and exit.
2. Else:  
   - Loop while `a >= 1 || b >= 1`:  
     a. If `a >= b`:  
        - `b += 1`  
        - `a -= 2`  
     b. Else:  
        - `a += 1`  
        - `b -= 2`  
     c. If `a <= 0 || b <= 0`:  
        - `c += 1`  
        - Break loop  
     d. Else:  
        - `c += 1`  
3. Print final value of `c`.

**Control Structures:**
- Conditional: `if (a == 1 && b == 1)`
- Loop: `while (a >= 1 || b >= 1)`
- Nested conditionals for adjusting `a`/`b` and checking for termination.

**Dependencies:**
- `java.util.Scanner`
- `java.io.PrintWriter`
- `java.util.Arrays` (unused in core logic)  

**Data Flow:**
- Input values modified in-place during iterations.
- Counter `c` tracks valid operations until a non-positive value occurs.
