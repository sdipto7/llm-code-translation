**Function Signatures:**
- `public static void main(String[] args)`  
- `private static int solve()`  
- `private static int readInt()`  

**Variables:**
- `K: int`  
- `ans: int`  
- `remainder: int`  

**Logic Flow:**
1. Read integer `K` from input via `readInt()`.  
2. If `K % 2 == 0` OR `K % 5 == 0`:  
   - Return `-1`.  
3. If `K % 7 == 0`:  
   - Set `K = K / 7`.  
4. Set `K = K * 9`.  
5. Initialize `ans = 1`.  
6. Compute `remainder = 10 % K`.  
7. While `remainder != 1`:  
   - Increment `ans` by `1`.  
   - Update `remainder = (remainder * 10) % K`.  
8. Return `ans`.  

**Control Structures:**  
- `if` conditions for divisibility checks (steps 2, 3).  
- `while` loop iterating until `remainder` equals `1` (step 7).  

**Input/Output:**  
- Input: `K` read via `BufferedReader` and `StringTokenizer`.  
- Output: Result printed via `PrintWriter` in `main`.  

**Dependencies:**  
- `java.io.*` for I/O operations.  
- `java.util.*` for `StringTokenizer`.  

**Data Types:**  
- All variables (`K`, `ans`, `remainder`) are `int`.
