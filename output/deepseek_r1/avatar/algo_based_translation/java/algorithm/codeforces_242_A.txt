**Function Signature:**  
public static void main(String[] args)  

**Variables:**  
- Scanner `in`  
- int `x`, `y`, `a`, `b`, `count`  

**Input Operations:**  
1. Read `x` via `in.nextInt()`  
2. Read `y` via `in.nextInt()`  
3. Read `a` via `in.nextInt()`  
4. Read `b` via `in.nextInt()`  

**Data Processing Logic:**  
1. Initialize `count` to 0  
2. Outer loop: Iterate `i` from `a` to `x` (inclusive), incrementing by 1  
   - Inner loop: Iterate `j` from `b` to `y` (inclusive), incrementing by 1  
     - If `i <= j`, skip to next iteration of inner loop  
     - Else, increment `count` by 1  
3. Print `count`  
4. Repeat outer loop structure: Iterate `i` from `a` to `x` (inclusive), incrementing by 1  
   - Repeat inner loop structure: Iterate `j` from `b` to `y` (inclusive), incrementing by 1  
     - If `i <= j`, skip to next iteration of inner loop  
     - Else, print `i` concatenated with `j`  

**Control Structures:**  
- Nested `for` loops for both counting and printing  
- `continue` statement when `i <= j` in both phases  

**Output Operations:**  
- Print total `count`  
- Print each valid `i j` pair on separate lines  

**Dependencies:**  
- `java.util.Scanner`
