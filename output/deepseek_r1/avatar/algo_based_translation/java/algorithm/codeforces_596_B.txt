**Function Signature:**
- `public static void main(String[] args)`

**Variables:**
- `Scanner scan` initialized with `System.in`
- `int n` initialized via `scan.nextInt()`
- `long[] arr` of size `n`, populated via loop
- `long answer` initialized with `Math.abs(arr[0])`

**Input/Output Operations:**
- Read `n` from standard input
- Read `n` integers into `arr` via loop
- Print `answer` to standard output

**Control Structures:**
- For loop: `for (int i = 0; i < n; i++)` to populate `arr`
- For loop: `for (int i = 1; i < n; i++)` to compute `answer`

**Logic Flow:**
1. Initialize `scan` as `Scanner(System.in)`
2. Read integer `n`
3. Declare `arr` as `long[n]`
4. Loop from `i = 0` to `n-1`:
   - Read integer into `arr[i]`
5. Set `answer` to absolute value of `arr[0]`
6. Loop from `i = 1` to `n-1`:
   - Add `|arr[i] - arr[i-1]|` to `answer`
7. Print `answer`

**Dependencies:**
- `java.util.Scanner`
- `java.lang.Math` (for absolute value operations)
