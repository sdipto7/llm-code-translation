**Function Signature:**
- `public static void main(String[] args) throws IOException`

**Variables:**
- `Scanner sc` (type `Scanner`)
- `int n` (initialized via `sc.nextInt()`)
- `int a = 0`, `int b = 0`, `int c = 0`
- `int x`, `int y`

**Libraries/Dependencies:**
- `java.util.*`
- `java.io.*`

**Logic Flow:**
1. Read integer `n` from standard input.
2. Initialize `a`, `b`, `c` to 0.
3. Loop `n` times:
   - Read integer from input and add to `a`.
4. Loop `n - 1` times:
   - Read integer from input and add to `b`.
5. Loop `n - 2` times:
   - Read integer from input and add to `c`.
6. Compute `x = a - b`.
7. Compute `y = b - c`.
8. Print `x` followed by `y` on separate lines.

**Control Structures:**
- Three `for` loops:
  - `for (int i = 0; i < n; i++)`
  - `for (int i = 0; i < n - 1; i++)`
  - `for (int i = 0; i < n - 2; i++)`

**Input/Output Operations:**
- Input: Sequential integers read via `sc.nextInt()`.
- Output: Two lines containing `x` and `y` via `System.out.println`.
