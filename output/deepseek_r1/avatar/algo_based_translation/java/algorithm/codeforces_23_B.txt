**Algorithm:**

**Function Signatures:**
- `public static void main(String[] args)`
- `public static int maxLeft(int n)`

**Variables:**
- `Scanner scan` (Scanner type)
- `int t` (integer)
- `int i` (integer, loop counter)
- `int n` (integer, per test case input)

**Control Structures:**
- `for (int i = 0; i < t; i++)`
- `n > 2 ? n - 2 : 0` (ternary conditional)

**Input/Output Operations:**
- Read integer `t` from standard input via `scan.nextInt()`
- Loop `t` times:
  - Read integer `n` from standard input via `scan.nextInt()`
  - Print result of `maxLeft(n)` via `System.out.println()`

**Logic Flow:**
1. Initialize `scan` as a `Scanner` for `System.in`
2. Read `t` from input
3. For each `i` from 0 to `t-1`:
   - Read `n` from input
   - Compute `maxLeft(n)` using ternary condition:
     - If `n > 2`, return `n - 2`
     - Else, return `0`
   - Print computed value

**Dependencies:**
- `java.util.Scanner` (imported)
