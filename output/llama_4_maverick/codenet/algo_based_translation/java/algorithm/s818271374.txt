### Algorithm

1. Import required libraries: 
   - java.util.*

2. Define a class: 
   - Main

3. Define a function: 
   - public static void main(String[] args)

4. Declare variables:
   - Scanner sc
   - int N
   - int count1
   - int count2
   - int i
   - int a

5. Execution logic:
   1. Create a new Scanner object `sc` to read from System.in.
   2. Read an integer `N` from the input using `sc.nextInt()`.
   3. Initialize `count1` and `count2` to 0.
   4. Loop from `i = 0` to `N-1` using a for loop.
      1. Read an integer `a` from the input using `sc.nextInt()`.
      2. Check if `a` is divisible by 4 using the condition `a % 4 == 0`.
         1. If true, increment `count1` by 1.
      3. Else, check if `a` is even and `a/2` is odd using the condition `a % 2 == 0 && (a/2) % 2 == 1`.
         1. If true, increment `count2` by 1.
   5. After the loop, check if `count1` is greater than or equal to `N/2`.
      1. If true, output "Yes" to the console using `System.out.println("Yes")`.
   6. Else, check if `((N/2 - count1) * 2 + N % 2)` equals `count2`.
      1. If true, output "Yes" to the console using `System.out.println("Yes")`.
   7. Else, output "No" to the console using `System.out.println("No")`.
