1. Import the required library: java.util.Scanner
2. Define a class: Main with a main method: 
   - `public static void main(String[] args)`
   - Parameters: `String[] args`
   - Return type: void
3. Declare a variable: `Scanner sc` and initialize it with `new Scanner(System.in)`
4. Declare an integer variable: `int n` and read its value from `sc.nextInt()`
5. Declare an integer array: `int[] array` of size `n`
6. Declare two integer variables: `int max` initialized to 0 and `int min` initialized to 10000
7. Iterate over the array using a for loop:
   - Loop condition: `int i = 0; i < array.length; i++`
   - Inside the loop:
     1. Read an integer value from `sc.nextInt()` and store it in `array[i]`
     2. Check if `max` is less than `array[i]`:
        - If true, update `max` with `array[i]`
     3. Check if `min` is greater than `array[i]`:
        - If true, update `min` with `array[i]`
8. After the loop, calculate the difference between `max` and `min`
9. Output the result using `System.out.println(max - min)`
10. Close the `Scanner` object `sc` using `sc.close()`
