1. Import the required library: java.util.*
2. Define a class: Main
3. Define a static method: main with return type void and parameter String[] args
4. Declare a variable: sc of type Scanner and initialize it with new Scanner(System.in)
5. Declare variables: A, B, C of type int
6. Read input from the user using sc.nextInt() and store it in A, B, C respectively
7. Check the condition: if (A == B && B == C && C == A)
8. If the condition is true, execute: System.out.println("Yes")
9. If the condition is false, execute: System.out.println("No")
