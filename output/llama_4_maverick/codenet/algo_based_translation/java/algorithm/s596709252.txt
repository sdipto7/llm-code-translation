1. Import the required Java library: java.util.*
2. Define a public class: Main
3. Define a public static method: main with parameter args of type String[] and return type void
4. Declare a variable sc of type Scanner and initialize it with a new Scanner object using System.in
5. Declare variables x, y, z of type int
6. Read an integer from the input using sc.nextInt() and store it in x
7. Read an integer from the input using sc.nextInt() and store it in y
8. Read an integer from the input using sc.nextInt() and store it in z
9. Close the Scanner object sc using sc.close()
10. Print the values of z, x, y to the console in the specified order, separated by spaces, using System.out.println()
11. The program terminates.
