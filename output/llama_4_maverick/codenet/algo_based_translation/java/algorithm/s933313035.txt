1. Import required libraries: java.util.Scanner
2. Define a class Main with a static nested class Node
3. Main class contains:
   - main method with parameter args of type String[]
   - Variable sc of type Scanner initialized with System.in
   - Variable m of type int initialized with sc.nextInt()
   - Variable tree of type Node initialized as null
4. Loop from 0 to m-1
   - Read a String command from sc
   - If command equals "insert"
     - Read a long key from sc
     - If tree is null
       - Initialize tree with a new Node
       - Set tree.value to key
     - Else
       - Call tree.insert(key)
   - Else
     - If tree is not null
       - Call tree.printInOrder()
       - Print a newline
       - Call tree.printPreOrder()
       - Print a newline
5. Node class contains:
   - Variables value of type long, left and right of type Node
   - Method insert with parameter newValue of type long and return type void
     - If newValue is less than value
       - If left is null
         - Initialize left with a new Node
         - Set left.value to newValue
       - Else
         - Call left.insert(newValue)
     - Else
       - If right is null
         - Initialize right with a new Node
         - Set right.value to newValue
       - Else
         - Call right.insert(newValue)
   - Method printInOrder with no parameters and return type void
     - If left is not null
       - Call left.printInOrder()
     - Print value preceded by a space
     - If right is not null
       - Call right.printInOrder()
   - Method printPreOrder with no parameters and return type void
     - Print value preceded by a space
     - If left is not null
       - Call left.printPreOrder()
     - If right is not null
       - Call right.printPreOrder()
