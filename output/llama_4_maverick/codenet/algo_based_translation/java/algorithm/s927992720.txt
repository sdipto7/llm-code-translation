1. Import required library: java.util.Scanner
2. Define class: Main
3. Define function: main with parameter args of type String[] and return type void
4. Declare variable sc of type Scanner and initialize it with new Scanner(System.in)
5. Enter infinite loop: while(true)
6. Inside loop:
   - Declare variable a of type int and read input from sc using nextInt()
   - Declare variable op of type String and read input from sc using next()
   - Declare variable b of type int and read input from sc using nextInt()
7. Check condition: if op equals "?" then break the loop
8. Check condition: if op equals "+" then print result of a + b to console
9. Check condition: if op equals "-" then print result of a - b to console
10. Check condition: if op equals "*" then print result of a * b to console
11. Check condition: if op equals "/" then print result of a / b to console (integer division)
12. Repeat steps 6-11 until condition in step 7 is met
