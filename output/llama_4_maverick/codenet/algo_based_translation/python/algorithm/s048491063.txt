1. Define a function `main()` with no parameters and no return type.
2. Read three integers `N`, `M`, and `Q` from standard input.
3. Initialize a 2D array `train` of size `N x N` with all elements set to 0.
4. Iterate `M` times:
   1. Read two integers `l` and `r` from standard input.
   2. Increment the value at `train[l-1][r-1]` by 1.
5. Initialize a 2D array `accum` of size `(N+1) x (N+1)` with the first row set to 0.
6. Iterate `N` times with index `i`:
   1. Initialize a temporary array `temp` of size `N+1` with the first element set to 0.
   2. Initialize a variable `v` to 0.
   3. Iterate `N` times with index `j`:
      1. Increment `v` by `train[i][j]`.
      2. Set `temp[j+1]` to `accum[i][j+1] + v`.
   4. Append `temp` to `accum`.
7. Iterate `Q` times:
   1. Read two integers `p` and `q` from standard input.
   2. Calculate `v = accum[q][q] - accum[p-1][q] - accum[q][p-1] + accum[p-1][p-1]`.
   3. Print `v` to standard output.
8. The program requires the ability to read from standard input and write to standard output.
