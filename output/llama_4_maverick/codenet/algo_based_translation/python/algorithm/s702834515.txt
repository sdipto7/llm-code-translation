1. Define a function `calculate_sum` with one integer parameter `n` and return type integer.
2. Initialize a variable `r` of type integer to 0.
3. Iterate over a range from 0 to `n-1` using a for loop with variable `ii`.
4. Inside the loop, check the condition `(ii+1) % 3 != 0` and `(ii+1) % 5 != 0` using a logical AND operator.
5. If the condition is true, add `(ii+1)` to `r`.
6. After the loop finishes, return the value of `r`.
7. Read an integer input from the user and store it in `n`.
8. Call the `calculate_sum` function with `n` as the argument and print the returned result.
