2. Declare variables `x`, `a`, `b`, and `result` of type `int`.
3. Read an integer input from the user and store it in `x`.
4. Read an integer input from the user and store it in `a`.
5. Read an integer input from the user and store it in `b`.

  4. `result: int = calculate_modulo(x, a, b)`

  1. Read an integer from the standard input.

  1. Calculate `(x - a) % b`.
  2. Return the calculated result.
