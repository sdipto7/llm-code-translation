3. Read input from the user using `input().split()` and store the values in `a`, `b`, `c` after converting them to `int` using `map(int, ...)`.
4. Evaluate the condition `a + b >= c`.
5. If the condition is `True`, return the string "Yes".
6. If the condition is `False`, return the string "No".

8. The required libraries are the built-in Python libraries, specifically no external modules are required.
