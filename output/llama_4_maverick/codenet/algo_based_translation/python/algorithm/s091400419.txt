5. Within `calculate_max_result`, create a list `results` containing the sum, difference, and product of `a` and `b`.

7. Define a `main` function with no parameters and no return value.
8. Within `main`, call `get_user_input` to obtain two integers `a` and `b`.
9. Call `calculate_max_result` with `a` and `b` and store the result.
10. Print the result obtained from `calculate_max_result`.
11. Call the `main` function to start the program execution.
