1. Define function `solve` with parameters `S`, `T`, `A`, `B`, `U`
   1.1. If `S` equals `U`, decrement `A` by 1
   1.2. If `T` equals `U`, decrement `B` by 1
   1.3. Print values of `A` and `B`

2. Define function `main`
   2.1. Define function `iterate_tokens`
      2.1.1. For each line in input
         2.1.1.1. For each word in line
            2.1.1.1.1. Yield word
   2.2. Initialize `tokens` with result of `iterate_tokens`
   2.3. Read `S`, `T`, `A`, `B`, `U` from `tokens`
      2.3.1. `S` = next token
      2.3.2. `T` = next token
      2.3.3. `A` = integer value of next token
      2.3.4. `B` = integer value of next token
      2.3.5. `U` = next token
   2.4. Call `solve` with `S`, `T`, `A`, `B`, `U`

3. If script is main program
   3.1. Call `main`
