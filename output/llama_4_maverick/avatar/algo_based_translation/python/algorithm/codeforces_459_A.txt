1. Read input and store it in 'given'
2. Split 'given' into a list 'l1' using space as the delimiter
3. Convert each element in 'l1' to an integer
4. Assign the first four elements of 'l1' to 'x1', 'y1', 'x2', 'y2' respectively
5. Calculate 'denominator' as 'x2 - x1' and 'numerator' as 'y2 - y1'
6. If 'denominator' is not equal to 0, calculate 'quotient' as 'numerator / denominator'
7. If 'numerator' equals 0, 
    a. Calculate 'd' as the integer square root of '(x2 - x1)^2 + (y2 - y1)^2'
    b. Assign 'x1' to 'x4' and 'x2' to 'x3'
    c. Calculate 'y3' as 'y2 + d' and 'y4' as 'y1 + d'
    d. Output 'x3', 'y3', 'x4', 'y4'
8. Else if 'denominator' equals 0, 
    a. Assign 'y2' to 'y4' and 'y1' to 'y3'
    b. Calculate 'd' as the integer square root of '(x2 - x1)^2 + (y2 - y1)^2'
    c. Calculate 'x4' as 'x1 + d' and 'x3' as 'x2 + d'
    d. Output 'x3', 'y3', 'x4', 'y4'
9. Else if 'quotient' equals 1, 
    a. Assign 'x2' to 'x4' and 'x1' to 'x3'
    b. Assign 'y1' to 'y4' and 'y2' to 'y3'
    c. Output 'x3', 'y3', 'x4', 'y4'
10. Else if 'quotient' equals -1, 
    a. Assign 'x1' to 'x4' and 'x2' to 'x3'
    b. Assign 'y2' to 'y4' and 'y1' to 'y3'
    c. Output 'x3', 'y3', 'x4', 'y4'
11. Otherwise, output '-1'
