1. Read the number N from input.
2. Read N integers, store them in list S, and sort S in descending order.
3. Initialize a flag list of size N with all elements set to True.
4. Initialize an empty list cur.
5. Append the first element of S to cur and set the corresponding flag to False.
6. For N iterations:
   6.1. Initialize j to 0 and jM to the current size of cur.
   6.2. Iterate through S:
      6.2.1. If the current element in S is flagged True and is less than the jth element in cur:
         6.2.1.1. Append the current element of S to cur.
         6.2.1.2. Increment j.
         6.2.1.3. Set the corresponding flag in S to False.
         6.2.1.4. If j equals jM, break the iteration.
   6.3. If the iteration completes without breaking (i.e., j never equals jM), return "No".
   6.4. Sort cur in descending order.
7. If the loop completes N iterations, return "Yes".
