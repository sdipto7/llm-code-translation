1. Read two integer inputs, n and m.
2. If n is less than 1, then
   2.1. If m is less than 1, then
      2.1.1. Output n and m.
      2.1.2. Terminate.
   2.2. Output "Impossible".
   2.3. Terminate.
3. If m is greater than n, then
   3.1. Set min to m.
   3.2. Set max to n + m - 1.
   3.3. Output min and max.
4. If m is less than or equal to n, then
   4.1. Set min to n.
   4.2. Set max to n + m - 1.
   4.3. If m equals 0, then
      4.3.1. Set max to n.
   4.4. Output min and max.
