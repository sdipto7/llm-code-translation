1. mod = 10^9 + 7
2. l = input string
3. n = length of l
4. Initialize dp as a 2D array of size (n+1) x 2 with all elements as 0
5. dp[0][0] = 1
6. For i from 0 to n-1:
   6.1. c = character at index i in l
   6.2. If c == '0':
        6.2.1. dp[i+1][0] = dp[i][0]
        6.2.2. dp[i+1][1] = dp[i][1] * 3
   6.3. Else:
        6.3.1. dp[i+1][0] = dp[i][0] * 2
        6.3.2. dp[i+1][1] = dp[i][0] + dp[i][1] * 3
   6.4. dp[i+1][0] = dp[i+1][0] mod mod
   6.5. dp[i+1][1] = dp[i+1][1] mod mod
7. result = (dp[n][0] + dp[n][1]) mod mod
8. Output result
