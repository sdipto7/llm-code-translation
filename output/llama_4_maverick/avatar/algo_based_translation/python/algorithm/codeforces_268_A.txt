1. READ n FROM INPUT
2. INITIALIZE EMPTY LIST q
3. INITIALIZE count TO 0
4. FOR i FROM 1 TO n
   4.1 READ a FROM INPUT AND SPLIT INTO LIST
   4.2 APPEND a TO q
5. FOR EACH j IN q
   5.1 FOR EACH k IN q
      5.1.1 IF k EQUALS j THEN CONTINUE TO NEXT k
      5.1.2 ELSE IF FIRST ELEMENT OF j EQUALS LAST ELEMENT OF k THEN INCREMENT count BY 1
      5.1.3 ELSE CONTINUE TO NEXT k
6. OUTPUT count
