1. Define function ok(last4)
   1.1 For i from 0 to 3
      1.1.1 Create a copy of last4 in t
      1.1.2 If i is greater than or equal to 1, swap t[i] and t[i-1]
      1.1.3 If the count of 'AGC' in the joined t is greater than or equal to 1, return False
   1.2 Return True

2. Define function dfs(cur, last3)
   2.1 If last3 is in memo[cur], return memo[cur][last3]
   2.2 If cur equals n, return 1
   2.3 Initialize ret to 0
   2.4 For each character c in "ACGT"
      2.4.1 If ok(last3 + c) is True, add dfs(cur + 1, last3[1:] + c) to ret modulo mod
   2.5 Store ret in memo[cur][last3]
   2.6 Return ret

3. Set n to the integer value of input
4. Set mod to 10^9 + 7
5. Initialize memo as a list of n+1 empty dictionaries
6. Call dfs(0, 'TTT') and print the result
