1. Read input values N, A, and B.
2. Check if 1 ≤ N ≤ 10^18, 0 ≤ A, 0 ≤ B, 0 < A + B, and A + B ≤ 10^18.
3. If the conditions in step 2 are met, proceed to step 4; otherwise, stop.
4. Check if A + B ≤ N.
5. If A + B ≤ N, calculate k = N / (A + B).
6. Calculate the remainder = N - k * (A + B).
7. If remainder > A, calculate result = A * k + A.
8. If remainder ≤ A, calculate result = A * k + remainder.
9. If A + B > N, check if A ≤ N.
10. If A ≤ N, result = A.
11. If A > N, result = N.
12. Output the calculated result.
