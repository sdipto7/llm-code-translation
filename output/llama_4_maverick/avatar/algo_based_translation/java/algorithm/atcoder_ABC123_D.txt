1. Read three integers into an array x.
2. Find the maximum value among the three integers read in step 1.
3. Read an integer k.
4. Create three lists a[0], a[1], a[2] to store long integers.
5. For each i from 0 to 2, read x[i] long integers into a[i] and sort a[i] in ascending order.
6. Create a list ans to store long integers.
7. For each i from 0 to x[0]-1, j from 0 to x[1]-1, and m from 0 to x[2]-1, check if (i+1)*(j+1)*(m+1) <= k.
8. If the condition in step 7 is true, calculate a[0].get(x[0]-i-1) + a[1].get(x[1]-j-1) + a[2].get(x[2]-m-1) and add it to ans.
9. Sort ans in ascending order.
10. Let n be the size of ans.
11. For i from 0 to k-1, print ans.get(n-i-1).
