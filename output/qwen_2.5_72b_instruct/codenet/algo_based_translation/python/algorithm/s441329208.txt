Algorithm:
1. Import the `sys` module to handle input and output operations.
2. Define `read`, `readline`, and `readlines` as aliases for `sys.stdin.read`, `sys.stdin.readline`, and `sys.stdin.readlines` respectively.
3. Set the recursion limit to 10,000,000 using `sys.setrecursionlimit`.
4. Read a line of input, strip any trailing whitespace, sort the characters in ascending order, and store the result in variable `s` of type list.
5. Read another line of input, strip any trailing whitespace, sort the characters in descending order, and store the result in variable `t` of type list.
6. Compare the two lists `s` and `t` lexicographically.
7. If `s` is lexicographically less than `t`, print "Yes".
8. Otherwise, print "No".
