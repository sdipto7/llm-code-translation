Function Signature: `multiply_and_process(input_string: str) -> int`

Variable Declarations:
- `A: int`
- `B: str`
- `B_processed: int`

Logic Flow:
1. Read input string from standard input
2. Split input string into two parts using space as delimiter
3. Assign first part to variable `A` and convert to integer
4. Assign second part to variable `B`
5. Remove all periods from string `B` and store result in `B_processed`
6. Convert `B_processed` to integer
7. Multiply `A` by `B_processed`
8. Perform integer division of the result by 100
9. Print the final result to standard output
