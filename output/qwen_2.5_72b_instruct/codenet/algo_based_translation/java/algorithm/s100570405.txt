Logic flow in execution order:
1. `main(String args[])` is called, creating a new instance of `Main` and calling `run()`.
2. Inside `run()`, `Sd` and `T` are read from input.
3. Initialize `count = 0`, `ok = false`, `first = -1`, `end = -1`.
4. Loop over `Sd` with `j` from 0 to `Sd.length() - 1`:
   - Reset `count = 0`.
   - Loop over `Sd` with `i` from 0 to `Sd.length() - j - 1`:
     - Set `c = Sd.charAt(i + j)`.
     - If `c == '?' || c == T.charAt(count)`:
       - Increment `count`.
       - If `count == T.length()`:
         - Set `first = i + j - T.length() + 1`, `end = i + j`.
         - Set `ok = true`.
         - Break inner loop.
     - Else:
       - Set `count = 0`.
5. If `ok`:
   - Loop over `Sd` with `i` from 0 to `Sd.length() - 1`:
     - If `first <= i && i <= end`:
       - Print `T.charAt(i - first)`.
     - Else:
       - If `Sd.charAt(i) == '?'`:
         - Print `"a"`.
       - Else:
         - Print `Sd.charAt(i)`.
   - Print newline.
6. Else:
   - Print `"UNRESTORABLE"`.
