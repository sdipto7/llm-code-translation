- Main function `main(String[] args)`
- Variable declarations: `Scanner sc`, `int n`, `int[] a`, `int ans`
- Input operations: `n = sc.nextInt()`, `a[i] = sc.nextInt()` in a loop from `i = 0` to `i < n`
- Logic flow:
  - Initialize `ans` to `0`
  - Loop from `i = 1` to `i < n-1`
    - Conditional: `if(a[i] != Math.min(a[i-1], Math.min(a[i], a[i+1])) && a[i] != Math.max(a[i-1], Math.max(a[i], a[i+1])))`
      - Increment `ans` by `1`
- Output operation: `System.out.println(ans)`
- Close scanner: `sc.close()`
- Required library: `java.util.*`
