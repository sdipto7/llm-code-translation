     - Increment swapCount by 1.

7. Loop from 1 to n-1:
   - Concatenate " " + a[i] to ans.
8. Print ans.
9. Print swapCount.

- Convert sorted array to space-separated string ans.
- Print sorted array ans.
- Print number of swaps swapCount.
