- `int ans` of type `int`

Logic flow:
1. Create a new `Scanner` object for reading input from standard input.
2. Read an integer `n` from the input.
3. Initialize an array `h` of size `n`.
4. For each index `i` from 0 to `n-1`, read an integer and store it in `h[i]`.
5. Initialize `ans` to 0.
6. Initialize `out` to 0.
7. For each index `i` from 0 to `h.length-1`:
   - If `out` is less than `h[i]`:
     - Add the difference between `h[i]` and `out` to `ans`.
     - Set `out` to `h[i]`.
   - Otherwise, set `out` to `h[i]`.
8. Print the value of `ans`.

- Read integers from standard input using `sc.nextInt()`.
- Print the final value of `ans` to standard output using `System.out.println(ans)`.
