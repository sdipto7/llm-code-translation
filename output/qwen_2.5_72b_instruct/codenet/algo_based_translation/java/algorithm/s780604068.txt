Logic Flow and Control Structures:
1. Read an integer `N` from standard input.
2. Initialize an array `map` of size `N * 2`.
3. For loop from `i = 0` to `i < N * 2` (inclusive):
   - Read an integer from standard input and store it in `map[i]`.
4. Sort the array `map`.
5. For loop from `i = 0` to `i < N * 2` (inclusive) with step size 2:
   - Add the minimum of `map[i]` and `map[i + 1]` to `max`.
6. Print the value of `max` to standard output.

Input/Output Operations:
- Read integer `N` from standard input.
- Read `N * 2` integers from standard input and store them in `map`.
- Output the value of `max` to standard output.
