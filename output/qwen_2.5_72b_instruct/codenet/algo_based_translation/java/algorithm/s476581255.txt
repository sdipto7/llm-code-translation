Logic flow:
1. Initialize `Scanner sc` to read input.
2. Read integer `n` from input.
3. Declare array `a` of size `n`.
4. For `i` from `0` to `n-1`, read integer and assign to `a[i]`.
5. Call `bubbleSort(a, n)` and store result in `count`.
6. For `i` from `0` to `n-1`, print `a[i]` followed by space if `i < n-1`, else print `a[i]` followed by newline.
7. Print `count`.

Function `bubbleSort(int[] a, int n) : int`
1. Initialize `flag` to `true`.
2. Initialize `count` to `0`.
3. While `flag` is `true`:
   - Set `flag` to `false`.
   - For `j` from `n-1` to `1`:
     - If `a[j] < a[j-1]`:
       - Assign `a[j]` to `free`.
       - Assign `a[j-1]` to `a[j]`.
       - Assign `free` to `a[j-1]`.
       - Increment `count` by `1`.
       - Set `flag` to `true`.
4. Return `count`.
