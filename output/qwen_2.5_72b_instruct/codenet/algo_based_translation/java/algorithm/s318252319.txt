- `public static void main(String[] args)`

**Variable Declarations and Data Types:**
- `Scanner sc` of type `Scanner`
- `String s` of type `String`
- `int n` of type `int`

**Logic Flow in Execution Order:**
1. Create an instance of `Scanner` to read input from standard input.
2. Read a `String` input from the user and assign it to `s`.
3. Extract the last character from `s`, convert it to an `int`, and assign it to `n`.
4. Check if `n` is equal to 3:
   - If true, print "bon".
5. If the previous condition is false, check if `n` is equal to 0, 1, 6, or 8:
   - If true, print "pon".
6. If none of the previous conditions are true, print "hon".
7. Close the `Scanner` instance.

**Control Structures:**
- `if (n == 3)`
- `else if (n == 0 || n == 1 || n == 6 || n == 8)`

**Input/Output Operations:**
- Read input using `sc.next()`.
- Print output using `System.out.println()`.

**Data Processing Logic:**
- Extract the last character of the string input.
- Convert the extracted character to an integer.
- Use conditional statements to determine the appropriate output based on the value of `n`.
