1. `n = in.nextInt()`
2. `m = in.nextInt()`
3. `bannedRows = new HashSet<>()`

5. For loop `for (int i = 0; i < m; i++)`

   - If `r > 1 && r < n` then `bannedRows.add(r)`
   - `c = in.nextInt()`
   - If `c > 1 && c < n` then `bannedCols.add(c)`
6. `answer = (n - 2) * 2`
7. `answer -= bannedRows.size()`
8. `answer -= bannedCols.size()`
9. If `n % 2 != 0`
   - `mid = (n + 1) / 2`
   - If `!bannedRows.contains(mid) && !bannedCols.contains(mid)` then `answer--`
10. `out.println(answer)`
11. `out.close()`
