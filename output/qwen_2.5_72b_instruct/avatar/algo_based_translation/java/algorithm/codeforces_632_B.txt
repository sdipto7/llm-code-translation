Function: main
Parameters: String[] args
Return Type: void

Variable Declarations:
- Scanner sc: new Scanner(System.in)
- int n: sc.nextInt()
- int[] a: new int[n]
- String s: sc.next()
- long sum: 0
- long ans: sum
- long sum1: sum

Logic Flow:
1. Initialize Scanner object for input
2. Read integer n from input
3. Declare array a of size n
4. For i from 0 to n-1:
   - Read integer from input and assign to a[i]
5. Read string s from input
6. Initialize sum to 0
7. For i from 0 to length of s - 1:
   - If character at position i in s is 'B':
     - Add a[i] to sum
8. Assign sum to ans
9. Assign sum to sum1
10. For i from 0 to length of s - 1:
    - If character at position i in s is 'A':
      - Add a[i] to sum1
    - Else:
      - Subtract a[i] from sum1
    - Update ans to maximum of ans and sum1
11. Assign sum to sum1
12. For i from length of s - 1 to 0:
    - If character at position i in s is 'A':
      - Add a[i] to sum1
    - Else:
      - Subtract a[i] from sum1
    - Update ans to maximum of ans and sum1
13. Print value of ans

Libraries:
- java.util.Scanner
