Function: getPoints
Parameters: int n, int k, int l, int r, int sAll, int sk
Return Type: int[]
Variables:
- int[] ans
- int idx

Logic Flow:
1. Initialize ans as an int array of size n
2. Fill ans array with l
3. Subtract sk + (n - k) * l from sAll
4. Subtract k * l from sk
5. While sk > 0
   a. Set idx to n - 1
   b. While sk > 0 and idx >= n - k
      i. Increment ans[idx]
      ii. Decrement idx
      iii. Decrement sk
6. While sAll > 0
   a. Set idx to 0
   b. While sAll > 0 and idx < n - k
      i. Increment ans[idx]
      ii. Increment idx
      iii. Decrement sAll
7. Return ans
