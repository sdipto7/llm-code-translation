Function Signature: `int solve()`

Variable Declarations:
- `int K`

- `int remainder = 10 % K`

- Read an integer `K` using `readInt()`

- While `remainder != 1`
  - Increment `ans`
  - Update `remainder = remainder * 10 % K`
