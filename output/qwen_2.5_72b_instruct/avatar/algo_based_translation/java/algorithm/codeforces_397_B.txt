Logic Flow:

1. Initialize scanner and printwriter for input and output respectively.
2. Read the number of test cases `t`.
3. For each test case:
   1. Read `n`, `a`, and `b`.
   2. If `n < a`, print "No" and continue to next test case.
   3. If `a == b`:
      1. If `n % a == 0`, print "Yes".
      2. Else, print "No".
      3. Continue to next test case.
   4. Calculate `x = b / (b - a)`.
   5. If `n > x * a`, print "Yes" and continue to next test case.
   6. Initialize `low` to 1, `high` to `x + 1`, and `ans` to 1.
   7. Perform binary search:
      1. While `low <= high`:
         1. Calculate `mid = (low + high) / 2`.
         2. If `mid * a < n`, set `low` to `mid + 1`.
         3. Else, set `ans` to `mid` and `high` to `mid - 1`.
   8. Check if `n > (ans - 1) * b` and `n < ans * a`:
      1. If true, print "No".
      2. Else, print "Yes".
4. Close the printwriter.
