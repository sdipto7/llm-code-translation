Function: main
Parameters: String[] args
Return Type: void

Variables:
- int n
- long k
- long[] arr
- HashSet<Long> set
- int count

Logic Flow:
1. Initialize Scanner object
2. Read integer n from input
3. Read long k from input
4. Initialize long array arr of size n
5. Initialize HashSet<Long> set
6. For i from 0 to n-1:
   - Read long from input and store in arr[i]
7. Sort array arr
8. Initialize count to 0
9. For i from 0 to n-1:
   - If arr[i] % k != 0:
     - Increment count
     - Add arr[i] to set
   - Else if set does not contain arr[i] / k:
     - Increment count
     - Add arr[i] to set
10. Print count
