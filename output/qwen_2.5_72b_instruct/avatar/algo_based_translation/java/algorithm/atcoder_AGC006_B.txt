Function signatures:
- `main(String[] args)`: void
- `print(int x, int n)`: void

Variable declarations and data types:
- `Scanner scanner`: Scanner
- `int n`: int
- `int x`: int
- `int max`: int
- `String sep`: String
- `String ans`: String

Logic flow in execution order:
1. Create a Scanner object to read input.
2. Read integer n from input.
3. Read integer x from input.
4. Call `print(x, n)`.

Function `print(int x, int n)`:
1. Calculate `max` as `n * 2 - 1`.
2. If `x` is equal to 1 or `x` is equal to `max`, print "No" and return.
3. Print "Yes".
4. Set `sep` to the system line separator.
5. Create a stream from `x + n - 1` to `x + n + max - 1`.
6. For each element `e` in the stream, calculate `(e % max + 1)` and convert it to a String, appending `sep`.
7. Collect all Strings into one String `ans` using StringBuilder.
8. Print `ans`.
