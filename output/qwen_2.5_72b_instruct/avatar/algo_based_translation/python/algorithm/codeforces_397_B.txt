- `__init__` (self): 
  - self.result: str
  - self.t: int
  - self.queries: list of list of int
- `read_input` (self): 
  - self.t = int(input())
  - for _ in range(self.t):
    - self.queries.append([int(x) for x in input().split(" ")])
- `process_task` (self): 
  - res: list of str
  - for query in self.queries:
    - k = query[0] // query[1]
    - if k * query[2] >= query[0]:
      - res.append("Yes")
    - else:
      - res.append("No")
  - self.result = "\n".join(res)
- `get_result` (self) -> str: 
  - return self.result
- main:
  - Solution = CodeforcesTask397BSolution()
  - Solution.read_input()
  - Solution.process_task()
  - print(Solution.get_result())
