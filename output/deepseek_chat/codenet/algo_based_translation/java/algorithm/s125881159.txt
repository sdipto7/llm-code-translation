**Function Signature:**
- `main(String[] args)`: void

**Variables:**
- `Scanner sc`: Scanner object for input
- `int n`: integer from input
- `int d`: integer initialized to 0
- `int i`: loop counter

**Logic Flow:**
1. Create Scanner object reading from standard input
2. Read integer `n` from input
3. Initialize `d` to 0
4. For `i` from 0 to `n-1`:
   - Read next integer from input
   - Calculate remainder when divided by 2
   - Add result to `d`
5. Check if `d % 2 == 0`
6. Print "YES" if true, "NO" if false

**Control Structures:**
- For loop: `for (int i = 0; i < n; i++)`
- Conditional: ternary operator checking `d % 2 == 0`

**Input/Output:**
- Input: Reads `n` integers from standard input
- Output: Prints "YES" or "NO" to standard output

**Dependencies:**
- `java.util.Scanner`
- `java.lang.System`
