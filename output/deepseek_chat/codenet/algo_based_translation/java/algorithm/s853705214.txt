**Function Signature:**
- main(String[] args): void

**Variables:**
- Scanner sc: Scanner object for input
- int n: integer input
- int[] nums: integer array of size n
- int i: loop counter
- int min: integer initialized to Integer.MAX_VALUE
- int countNegatives: integer initialized to 0
- long sum: long integer initialized to 0
- int num: loop variable for array elements
- int abs: absolute value of num

**Logic Flow:**
1. Create Scanner object reading from System.in
2. Read integer n from input
3. Initialize integer array nums of size n
4. For i from 0 to n-1:
   - Read integer from input and store in nums[i]
5. Initialize min to Integer.MAX_VALUE
6. Initialize countNegatives to 0
7. Initialize sum to 0
8. For each num in nums:
   - Compute abs = absolute value of num
   - Add abs to sum
   - Set min to minimum of min and abs
   - If num < 0: increment countNegatives by 1
9. If countNegatives % 2 == 0:
   - Print sum
   Else:
   - Print sum - (min * 2)

**Control Structures:**
- for loop: for (int i = 0; i < n; i++)
- enhanced for loop: for (int num : nums)
- conditional: if (num < 0)
- ternary conditional: countNegatives % 2 == 0 ? sum : sum - (min*2)

**Input/Output Operations:**
- Input: Read integer n, then n integers
- Output: Print computed result

**Required Libraries:**
- java.util.Scanner
