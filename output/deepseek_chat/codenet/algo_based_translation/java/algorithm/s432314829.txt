**Function Signature:**
- `main(String[] args): void`

**Variables:**
- `Scanner stdIn = new Scanner(System.in)`
- `int n = stdIn.nextInt()`
- `int m = stdIn.nextInt()`
- `int[] dp = new int[n + 2]`

**Logic Flow:**
1. Initialize `dp[n + 1] = 0`
2. Initialize `dp[n] = 1`
3. For `i` from `0` to `m - 1`:
   - Read integer `x = stdIn.nextInt()`
   - Set `dp[x] = -1`
4. For `i` from `n - 1` down to `0`:
   - If `dp[i] == -1`:
     - Set `dp[i] = 0`
   - Else:
     - Set `dp[i] = (dp[i + 1] + dp[i + 2]) % 1000000007`
5. Print `dp[0]`

**Control Structures:**
- For loop: `for (int i = 0; i < m; i++)`
- For loop: `for (int i = n - 1; i >= 0; i--)`
- Conditional: `if (dp[i] == -1)`

**Input/Output:**
- Input: `n`, `m`, followed by `m` integers
- Output: Print `dp[0]`

**Dependencies:**
- `java.util.Scanner`
