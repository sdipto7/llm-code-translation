Algorithm:
- Import java.util.Scanner
- Define class Main
- Define main method with String[] args parameter
  - Create Scanner object sc reading from System.in
  - Call solve method passing sc
  - Close scanner automatically via try-with-resources
- Define static solve method with Scanner sc parameter
  - Read next token from scanner and convert to char array a
  - Check if a[0] equals a[1] OR a[1] equals a[2] OR a[2] equals a[0]
    - If true: print "No"
    - Else: print "Yes"
