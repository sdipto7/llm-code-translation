**Function Signature:**
- main(String[] args): void

**Variables:**
- Scanner sc: Scanner object for input
- int n: integer input
- int[] l: integer array of size n
- int[] r: integer array of size n
- int sum: integer initialized to 0
- int i: loop counter

**Logic Flow:**
1. Create Scanner object to read from standard input
2. Read integer n from input
3. Initialize integer array l of size n
4. Initialize integer array r of size n
5. For i from 0 to n-1:
   - Read integer into l[i]
   - Read integer into r[i]
6. Initialize sum to 0
7. For i from 0 to n-1:
   - Add (r[i] - l[i] + 1) to sum
8. Print sum

**Control Structures:**
- For loop: i = 0; i < n; i++ (executes n times)
- For loop: i = 0; i < n; i++ (executes n times)

**Input/Output:**
- Input: integer n, followed by n pairs of integers
- Output: single integer sum

**Dependencies:**
- java.util.Scanner
