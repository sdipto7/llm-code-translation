**Function Signature:**
- main(String[] args): void

**Variables:**
- Scanner sc
- int N
- long[] a
- int tmp
- int ans
- int z
- int j
- int i

**Libraries/Dependencies:**
- java.util.Scanner
- java.util.Arrays

**Algorithm:**
1. Initialize Scanner with System.in
2. Read integer N from input
3. Declare array a of size N with type long
4. Initialize ans to 0
5. For z from 0 to N-1:
   - Read next integer into a[z]
6. Sort array a using Arrays.sort
7. If N equals 1 AND a[0] greater than 1:
   - Set ans to 1
8. Else:
   - For j from 0 to N-1:
     - If j equals 0 OR a[j] not equal to a[j-1]:
       - Set tmp to 1
       - For i from j+1 to N-1:
         - If a[j] equals a[i]:
           - Increment tmp by 1
         - Else:
           - Break inner loop
       - If tmp greater than a[j]:
         - Add (tmp - a[j]) to ans
       - Else if tmp less than a[j]:
         - Add tmp to ans
9. Print ans
