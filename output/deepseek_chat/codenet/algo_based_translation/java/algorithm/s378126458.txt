**Function Signature:**
- `main(String[] args)`: void

**Variables:**
- `Scanner sc`: Scanner object for input
- `int N`: integer, length of string
- `int Q`: integer, number of queries
- `String s`: input string
- `int[] pre`: integer array of size N+1
- `int l`: integer, query start index
- `int r`: integer, query end index

**Logic Flow:**
1. Initialize Scanner `sc`
2. Read integer `N`
3. Read integer `Q`
4. Read string `s`
5. Create integer array `pre` with size N+1
6. For `i` from 1 to N-1:
   - If character at position `i-1` in `s` is 'A' AND character at position `i` in `s` is 'C':
     - Set `pre[i]` to 1
7. For `i` from 1 to N-1:
   - Add `pre[i-1]` to `pre[i]`
8. While `Q` > 0:
   - Read integer `l`
   - Read integer `r`
   - Decrement `r` by 1
   - Calculate `pre[r] - pre[l-1]`
   - Print result
   - Decrement `Q` by 1

**Control Structures:**
- For loop: `for(int i = 1; i < N; i++)`
- While loop: `while(Q-- > 0)`
- Conditional: `if(s.charAt(i-1) == 'A' && s.charAt(i) == 'C')`

**Input/Output:**
- Input: Reads integers N, Q and string s from standard input
- Output: Prints integer results for each query

**Dependencies:**
- `java.util.Scanner`
