Algorithm:
1. Read four integers from input: N, A, B, C
2. Read N integers from input and store in list l
3. Define function dfs with parameters: cur (integer), a (integer), b (integer), c (integer) returning integer
4. If cur equals N:
   - If min(a, b, c) > 0: return (abs(A - a) + abs(B - b) + abs(C - c) - 30)
   - Else: return 1000000000
5. Else:
   - no_add = dfs(cur + 1, a, b, c)
   - add_a = dfs(cur + 1, a + l[cur], b, c) + 10
   - add_b = dfs(cur + 1, a, b + l[cur], c) + 10
   - add_c = dfs(cur + 1, a, b, c + l[cur]) + 10
   - Return minimum of no_add, add_a, add_b, add_c
6. Call dfs(0, 0, 0, 0) and print result
