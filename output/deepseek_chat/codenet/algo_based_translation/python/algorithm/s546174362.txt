**Function Signature:**
- No explicit function defined; main script execution

**Variables and Data Types:**
- N: integer (first input value)
- M: integer (second input value)
- A: list of integers (input list of numbers)
- i: integer (temporary index variable)

**Input Operations:**
- Read first line: split into two integers N and M
- Read second line: split into list of integers for A

**Logic Flow:**
1. Sort list A in ascending order
2. While M > 0:
   - Divide last element of A by 2 (integer division)
   - Decrement M by 1
   - Set i = -2
   - While M > 0 AND i >= -N AND A[i] >= A[-1]:
     - Divide element at index i by 2 (integer division)
     - Decrement M by 1
     - Decrement i by 1
3. Sort list A
4. Calculate and output sum of all elements in A

**Control Structures:**
- While loop: condition M > 0
- Nested while loop: conditions M > 0 AND i >= -N AND A[i] >= A[-1]
- List sorting operations

**Output Operation:**
- Print sum of all elements in list A

**Dependencies:**
- Standard Python input/output functions
- No external libraries required
