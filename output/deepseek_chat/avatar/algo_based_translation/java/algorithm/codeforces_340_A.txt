**Algorithm Steps:**
1. Read four integers: `x`, `y`, `a`, `b`
2. Set `c` to maximum of `x` and `y`
3. If `c > b`:
   - Print "0"
   - Set `ans = false`
4. While `ans` is true and `c` is not divisible by `min(x,y)`:
   - If `x > y`:
        - Set `c = x * j`
        - Increment `j`
   - Else:
        - Set `c = y * j`
        - Increment `j`
   - If `c > b`:
        - Print "0"
        - Set `ans = false`
        - Break loop
5. If `ans` is true:
   - Find first `i` in range `[a, b]` where `i % c == 0`
   - If such `i` exists:
        - Print `((b - i) / c) + 1`
   - Else:
        - Print "0"
