**Function Signatures:**
- `revpos(int[] p, int[] q) → int[]`
- `main(String[] args) → void`

**Variables and Data Types:**
- `Scanner scan`
- `int[] input` (size 4)
- `int[][] position` (4×2 array)
- `int[] output` (size 4)
- `int[] vec` (size 2)
- `int tmp`
- `int[] newpos` (size 2)

**Logic Flow:**
1. Create Scanner object for input
2. Initialize input array of 4 integers
3. Read 4 integers from stdin into input array
4. Close scanner
5. Initialize 4×2 position array
6. Set position[0] = [input[0], input[1]]
7. Set position[1] = [input[2], input[3]]
8. Call revpos(position[0], position[1]) → store result in position[2]
9. Call revpos(position[1], position[2]) → store result in position[3]
10. Check if position[0] equals revpos(position[2], position[3])
    - If true: print "OK"
11. Initialize output array of 4 integers
12. Set output[0] = position[2][0]
13. Set output[1] = position[2][1]
14. Set output[2] = position[3][0]
15. Set output[3] = position[3][1]
16. Print each element of output array

**revpos Function Logic:**
1. Initialize vec array of 2 integers
2. For i from 0 to 1:
    - vec[i] = p[i] - q[i]
3. Set tmp = vec[0]
4. Set vec[0] = vec[1]
5. Set vec[1] = -tmp
6. Initialize newpos array of 2 integers
7. For i from 0 to 1:
    - newpos[i] = q[i] + vec[i]
8. Return newpos

**Control Structures:**
- For loops: `for (int i = 0; i < length; i++)`
- Conditional: `if (position[0] == revpos(position[2], position[3]))`

**Input/Output Operations:**
- Input: Read 4 integers from stdin
- Output: Print "OK" (conditionally) and 4 integers

**Required Libraries:**
- `java.util.Scanner`
