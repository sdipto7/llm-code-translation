**Function Signature:**
- `main(String[] args)`: void

**Variables:**
- `Scanner in`: Scanner object for input
- `PrintWriter pw`: PrintWriter object for output
- `int N`: integer from input
- `int[] A`: integer array of size N
- `int n`: loop index
- `int min`: integer minimum value
- `int max`: integer maximum value
- `long mins`: long count of minimum values
- `long maxs`: long count of maximum values
- `int value`: loop variable for array traversal

**Libraries/Dependencies:**
- java.io.*
- java.util.*

**Algorithm Steps:**
1. Initialize Scanner with System.in
2. Initialize PrintWriter with System.out
3. Read integer N from input
4. Create integer array A of size N
5. For n from 0 to N-1:
   - Read integer from input and store in A[n]
6. Sort array A using Arrays.sort()
7. Set min = A[0]
8. Set max = A[N-1]
9. Initialize mins = 0
10. Initialize maxs = 0
11. For each value in array A:
    - If value equals min: increment mins by 1
    - If value equals max: increment maxs by 1
12. If min equals max:
    - Print (max - min) and (mins * (mins - 1) / 2)
13. Else:
    - Print (max - min) and (mins * maxs)
14. Close PrintWriter
15. (Optional debug method not called in main)
