**Function Signatures:**
- `public static void main(String[] args)`
- `public static int gcd(int a, int b)`

**Variables:**
- `Scanner sc`
- `int N`
- `int[] A` (static)
- `int[] M`
- `int[] L`
- `int[] R`
- `int answer`
- `int tmp`
- `int r`

**Logic Flow:**
1. Create Scanner object to read from standard input
2. Read integer N
3. Initialize array A of size N
4. For i from 0 to N-1: Read A[i]
5. Close Scanner
6. Initialize arrays M, L, R of size N
7. Set L[0] = A[0]
8. For i from 1 to N-1: L[i] = gcd(L[i-1], A[i])
9. Set R[N-1] = A[N-1]
10. For i from N-2 down to 0: R[i] = gcd(R[i+1], A[i])
11. Set answer = max(L[N-2], R[1])
12. For i from 1 to N-2: answer = max(answer, gcd(L[i-1], R[i+1]))
13. Print answer

**gcd Function:**
1. If b > a: swap a and b
2. If b == 0: return a
3. Set r = -1
4. Do:
   - r = a % b
   - a = b
   - b = r
   While r != 0
5. Return a

**Dependencies:**
- `java.util.Scanner`
