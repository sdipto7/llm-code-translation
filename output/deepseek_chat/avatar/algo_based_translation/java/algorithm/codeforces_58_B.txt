**Function Signature:**
- `main(String[] args) throws Exception`

**Variables:**
- `Scanner sc = new Scanner(System.in)`
- `int n = sc.nextInt()`
- `int m = n`

**Logic Flow:**
1. Read integer `n` from standard input
2. Print `n` followed by a space
3. Initialize `m` to `n`
4. Loop from `i = n-1` down to `1` (inclusive):
   - If `m` is divisible by `i` (`m % i == 0`):
     - Print `i` followed by a space
     - Set `m` to `i`
5. End loop when `i` reaches 0

**Control Structures:**
- `for (int i = n-1; i > 0; i--)`
- `if (m % i == 0)`

**Input/Output:**
- Input: Single integer from stdin
- Output: Sequence of integers separated by spaces

**Dependencies:**
- `java.util.Scanner`
- `java.lang.Exception`
