**Function Signature:**
- `main(String[] args)`: void return type

**Variables:**
- `Scanner scan`: Scanner object for input
- `int n`: input integer
- `int N = 100005`: constant (unused)
- `long[] cnt = new long[21]`: array of 21 long integers
- `long res = 0`: result accumulator

**Logic Flow:**
1. Initialize Scanner with System.in
2. Read integer n from input
3. Declare constant N (unused)
4. Initialize cnt array with 21 elements
5. For i from 0 to n-1:
   - Read integer x from input
   - Increment cnt[x + 10] by 1
6. Initialize res to 0
7. For i from 0 to 9:
   - Add cnt[i] * cnt[20 - i] to res
8. Add (cnt[10] * (cnt[10] - 1)) / 2 to res
9. Print res

**Control Structures:**
- For loop: `for (int i = 0; i < n; i++)`
- For loop: `for (int i = 0; i < 10; i++)`

**Input/Output:**
- Input: First integer n, then n integers
- Output: Single long integer result

**Dependencies:**
- java.util.Scanner
