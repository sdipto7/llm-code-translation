Algorithm:
- Function: main
  - Parameters: String[] args
  - Return type: void
- Variables:
  - Scanner sc = new Scanner(System.in)
  - int N = sc.nextInt()
  - int K = sc.nextInt()
  - int[] S = new int[N]
  - int temp = 0
  - int ans = 1000000000
- Execution flow:
  1. Read integer N from input
  2. Read integer K from input
  3. Initialize integer array S of size N
  4. For i from 0 to N-1:
     - Read integer S[i] from input
  5. If K == 1:
     - For i from 0 to N-1:
       - Set temp = S[i]
       - Set ans = minimum of absolute value of temp and current ans
     - Print ans
  6. Else if N - K != 0:
     - For i from 0 to N - K:
       - Set min = S[i]
       - Set max = S[i + K - 1]
       - If min < 0 and max > 0:
         - Set temp = minimum of (2 * (-min) + max) and ((-min) + 2 * max)
       - Else:
         - Set temp = maximum of absolute value of min and absolute value of max
       - Set ans = minimum of ans and temp
     - Print ans
  7. Else:
     - Set min = S[0]
     - Set max = S[N - 1]
     - If min < 0 and max > 0:
       - Print minimum of (2 * (-min) + max) and ((-min) + 2 * max)
     - Else:
       - Print maximum of absolute value of min and absolute value of max
  8. Close Scanner sc
- Required libraries: java.util.Scanner
