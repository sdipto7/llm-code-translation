**Function Signature:**
- `main(String[] args)`: void

**Variables:**
- `sc`: Scanner (input stream)
- `N`: int (number of sweets)
- `x`: long (available candy amount)
- `sweet`: List<Long> (sweet quantities)
- `a`: long (temporary sweet value)
- `num`: int (count of distributed sweets)

**Libraries/Dependencies:**
- java.util.*

**Algorithm:**
1. Initialize Scanner `sc` with System.in
2. Read integer `N` from input
3. Read long `x` from input
4. Create empty ArrayList `sweet` of Long type
5. For `i` from 0 to N-1:
   - Read long `a` from input
   - Add `a` to `sweet`
6. Sort `sweet` in ascending order
7. Initialize `num` to 0
8. For `i` from 0 to N-1:
   - If `x - sweet.get(num) >= 0`:
     - Subtract `sweet.get(num)` from `x`
     - Increment `num` by 1
   - Else: break loop
9. If `num == N` AND `x > 0`:
   - Decrement `num` by 1
10. Print `num`
