10. For i from arr.length to level3-1:
    - Read integer and store in arr2[i]
11. Initialize empty integer array arr3 of size n
12. For i from 0 to n-1:
    - Increment j by 1
    - Set arr3[i] = j
13. For i from 0 to n-1:
    - For x from 0 to arr2.length-1:
        - If arr3[i] equals arr2[x]:
