**Function Signatures:**
- `public static void main(String[] args)`
- `public static long getGCD(long a, long b)`

**Variable Declarations:**
- `Scanner sc` (Scanner object)
- `int n`
- `int m`
- `int[] a` (size n)
- `long gcd`
- `long lcd` (initialized to 1)

**Logic Flow:**

1. Initialize Scanner with System.in
2. Read integer n from input
3. Read integer m from input
4. Initialize integer array a of size n
5. For i from 0 to n-1:
   - Read integer from input
   - Divide integer by 2
   - Store result in a[i]
6. For i from 0 to n-1:
   - Calculate gcd = getGCD(lcd, a[i])
   - Update lcd = (lcd * a[i]) / gcd
   - If lcd > m:
        Print 0
        Return from main
7. For i from 0 to n-1:
   - Calculate (lcd / a[i]) % 2
   - If result equals 0:
        Print 0
        Return from main
8. Calculate result = (m / lcd + 1) / 2
9. Print result

**getGCD Function Logic:**
- If b == 0: return a
- Else: return getGCD(b, a % b)

**Control Structures:**
- For loops: `for (int i = 0; i < n; i++)`
- Conditional: `if (lcd > m)`
- Conditional: `if ((lcd / a[i]) % 2 == 0)`
- Recursion in getGCD

**Input/Output Operations:**
- Input: Reads integers from standard input
- Output: Prints 0 or calculated result to standard output

**Required Libraries:**
- java.util.Scanner
