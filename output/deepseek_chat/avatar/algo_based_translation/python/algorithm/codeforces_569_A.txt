- No explicit function defined; main script execution.

**Variables:**
- `t`: integer (input)
- `s`: integer (input)
- `q`: integer (input)
- `ans`: integer (initialized to 0)

**Input:**
- Read one line from standard input.
- Split the input string by whitespace.
- Convert each split substring to integer.
- Assign first integer to `t`, second to `s`, third to `q`.

**Logic Flow:**
1. Initialize `ans` to 0.
2. While `s` is less than `t`:
   - Multiply `s` by `q`.
   - Increment `ans` by 1.
3. Print `ans`.

**Control Structures:**
- While loop: condition `s < t`.

**Output:**
- Print integer `ans` to standard output.

- None (uses built-in functions).
