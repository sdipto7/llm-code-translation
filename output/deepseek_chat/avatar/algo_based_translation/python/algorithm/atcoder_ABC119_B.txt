Read integer N from standard input
Declare list x of type float with size N
For i from 0 to N-1:

    Replace all occurrences of 'JPY' with '* 1.0' in the string
    Replace all occurrences of 'BTC' with '* 380000.0' in the string

    Store result in x[i]
Calculate sum of all elements in x
