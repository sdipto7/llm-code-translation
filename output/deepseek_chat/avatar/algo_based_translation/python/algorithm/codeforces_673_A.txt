**Function Signature:**
- main() -> None

**Variables:**
- n: int
- arr: List[int]

**Input Operations:**
- Read integer n from stdin
- Read next line, split into list of integers for arr

**Logic Flow:**
1. Check if first element of arr > 15
   - If true: print 15 and terminate
2. Else: iterate through arr starting from index 1 to end
   - For each index i, check if arr[i] - arr[i-1] > 15
     - If true: print arr[i-1] + 15 and break loop
3. If loop completes without break:
   - Check if last element equals 90 OR last element + 15 >= 90
     - If true: print 90
     - Else: print last element + 15

**Control Structures:**
- If-else condition: arr[0] > 15
- For loop: range(1, len(arr))
- If condition inside loop: arr[i] - arr[i-1] > 15
- Else clause for loop completion
- Nested if-else: arr[-1] == 90 OR arr[-1] + 15 >= 90

**Output Operations:**
- Print integer results to stdout

**Dependencies:**
- None beyond standard input/output operations
